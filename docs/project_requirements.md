# 毕业设计课题任务与论文约束 (最高优先级)

## 课题软件任务
1. 实现人体肌电信号 (EMG) 的噪声分析与滤波降噪。
2. 实现 EMG 信号的分割与活动段提取。
3. 使用机器学习方法实现手势识别。
4. 使用深度学习方法实现手势识别。
5. 对各种识别方法的准确率与优缺点进行比较分析。

## 预期成果或目标
- 基于 EMG 信号实现手势识别软件。
- 同时使用机器学习与深度学习方法，覆盖十个以上手势。
- 识别准确率不低于 85% (主结果应满足此目标)。

## 论文内容约束
- 论文必须覆盖：噪声分析与滤波、分割与活动段提取、机器学习基线、深度学习方法、方法比较分析、最终系统结论。
- 中期报告中的基础证据必须保留并进入最终论文：DB5 数据集与滑窗处理、TD/FD/AR 特征、LDA/SVM/RF 基线、噪声注入与鲁棒性分析、feature ablation、clean vs noisy 对比。

## 论文消费约束 (用于自动写作)
- 不能将传统 ML、噪声/鲁棒性、feature ablation 视为历史废弃内容。
- 主结果必须来自 subject-dependent + LORO + repetition-level aggregation (非单个固定 split)。
- cross-subject 与 calibration 属于补充实验层，不得覆盖主结果。
